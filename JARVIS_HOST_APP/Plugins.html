Plugins

JarvisAssistant's functionality is powered by a modular plugin architecture. Explore the list of available plugins below.

Summarizer Plugin

Utilizes advanced algorithms to provide concise summaries of text content.

OS Automation Plugin

A core plugin that provides the ability to control and automate your operating system.

Screen Analyzer

Enables the assistant to analyze and provide help based on your live screen content.

Code Editor Plugin

Allows for quick access and control over various code editors.

Smart Home Plugin

Integrates with your smart home devices to control them with simple commands.
